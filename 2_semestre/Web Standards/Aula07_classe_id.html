BIG DATA

Big data é a área do conhecimento que estuda como tratar, analisar e obter informações a partir de conjuntos de dados muito grandes. O termo big data surgiu em 1997, e foi inicialmente utilizado para nomear conjuntos de dados não ordenados em rápido crescimento.

Inteligência artificial é toda tecnologia digital com capacidade para simular o modelo humano de raciocínio para solucionar as mais variadas questões. Para isso, elas são alimentadas por Big Data (dados em massa) e métodos avançados de aprendizado de máquina.
